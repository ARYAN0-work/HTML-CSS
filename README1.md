# Amazon Homepage Clone (HTML Structure Phase)

## 📌 Project Overview

This project is a structural clone of the Amazon homepage built using semantic HTML.

The primary focus of this phase is:

- Clean HTML architecture
- Proper semantic layout
- Functional block separation
- Production-style project organization

CSS styling will be implemented after the structural foundation is completed.

---

## 🎯 Project Goals

- Build the entire page structure before styling
- Practice section-based development
- Develop production-level thinking
- Maintain clean and meaningful Git commits
- Prepare structure for scalable CSS implementation

---

## 🏗 Architecture Planning

The page is divided into three major semantic sections:

### 1️⃣ Header
Contains two navigation layers:

- **Top Navigation**
  - Logo
  - Delivery location
  - Search bar (internally divided into 3 elements)
  - Account section
  - Orders section
  - Cart section

- **Bottom Navigation**
  - Menu options (e.g., All, Deals, Customer Service, etc.)

Each major navigation item is treated as a functional block.
Composite elements (like search and account sections) are structured with internal hierarchy.

---

### 2️⃣ Main
Contains:

- Hero section
- Product grid section

Each section is separated logically for scalability and clarity.

---

### 3️⃣ Footer
Will contain:

- Navigation links
- Informational links
- Copyright section

---

## 🧠 Development Approach

- Semantic HTML first
- Section-by-section implementation
- Clear hierarchy before styling
- No unnecessary div nesting
- CSS added only after structural clarity
- Small, meaningful Git commits
